# terraform_vpc_ec2_volume_webserver
## prerequisite
- terraform 0.12 versio requiment
- aws cli configure

# How To Use
- first command install plugin of terraform
```
terraform init
```
- second check the plan
```
terraform plan
```
- after check plan we are apply our terraform script

```
terraform apply
```
